# Dynamisches Programmieren

> Lösen eines Problems durch Aufteilung in Teilprobleme und systematische Speicherung von Zwischenresultaten.

## Elemente

1. Optimale Teilstruktur: Eine optimale Lösung des Gesamtproblems beinhaltet eine optimale Lösung der Teilprobleme.
1. Überlappende Teilprobleme: Teilprobleme bestehen aus gemeinsamen Teilproblemen.
1. Memorieren: Speichere Lösung von Teilproblemen in einer Tabelle.
